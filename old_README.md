# Workshop Primeiro Agente de IA

> **Workshop de desenvolvimento. Conexões com a OpenAI serão testadas via API, habilitando uma estrutura simples de resposta via prompt. Cópia e adaptação de 2 códigos python presentes em https://github.com/caio-moliveira/ai-engineer-roadmap/tree/main/03-ai-agents/02-my-first-agent. Agentes de IA não serão utilizados para codar, apenas para verificar e adequar o desenvolvimento após o uso, se necessário.**
